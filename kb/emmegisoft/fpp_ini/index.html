Configuración de Archivos INI

El archivo de configuración FP Pro.ini puede estar ubicado en cualquier carpeta a la que el usuario tenga acceso, pero por seguridad debería estar ubicado en la carpeta del usuario. Cuando el archivo se encuentra en la carpeta del usuarios Windows se encarga de la seguridad del mismo, lo cual es de importancia cuando se instala en Terminal Server.

Multi-Usuario (Terminal Server)

Esta es una configuración típica para terminal server, pero tambien es necesario cuando el mismo computador es usado por varios usuarios que requieren acceso a FP Pro.

[GENERAL]
LastUserFilter=xuserx
TMP_PATH=C:\USERS\xuserx\APPDATA\LOCAL\EMMEGISOFT\FP_PRO\TEMP\
LOG_PATH=C:\USERS\xuserx\APPDATA\LOCAL\EMMEGISOFT\FP_PRO\LOGS\

Exportación de maquinado y corte

Estas opciones pueden configurarse tambien desde la interfaz de usuario de FP Pro, pero es conveniente saber donde se encuentran para der modificadas manualmente.

[General]
CUTTING_LIST_MODE=3
CUTTING_LIST_ORDER=1
CUTTING_LIST_PIECES_SORTING=1
CUTTING_LIST_REFERENCE=5
