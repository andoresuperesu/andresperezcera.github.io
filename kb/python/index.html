Overview

Mi interés en Python nace con ayuda de Emilio Borelli y Jose Luis Caro. Ha sido un camino lento pero cada vez mas interesante hacia dominar este lenguaje (todavía no hago nada interesante). La meta es crear algunos proyectos los cuales espero enumerar en orden de complejidad. Esta se torna —entonces— en una manera de disipar el humo de forzarme a ejecutar esa tarea consistentemente.
